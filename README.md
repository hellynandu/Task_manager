# Task Manager Application

## Description
A simple task management web application built with Node.js, Express, and MongoDB.

## Features
- Add, edit, and delete tasks
- Mark tasks as complete
- Search functionality
- Filter by status
- Task statistics

## Technologies Used
- Node.js
- Express.js
- MongoDB
- EJS (Template Engine)
- CSS
